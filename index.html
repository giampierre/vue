# Guía Vue Framework

# ¿Qué es Vue?

Es un Framework progresivo, es decir, es un Framework que sirve para consumir interfaz del usuario.

Podemos Utilizar Vue como una librería copiando su cdn en nuestro proyecto o utilizarlo como un framework completo Vue Cli creando toda la estrucutura de un proyecto como lo haremos en este caso.

# Instalación

  1. Tener instalado Node

TIP

Node es un entorno de ejecución para instalar paquetes de JavaScrip

Descargar Node

Verificar la version que se instaló con este comando:

$ node --version
  1. Una vez teniendo instalado Node, procedemos a instalar vue cli con el siguiente comando:
$ npm install -g @vue/cli

TIP

Esto instalará vue de manera global en su computadora

Verificar la version que esta instalado con este comando:

$ vue --version

# Creando Proyecto

WARNING

Exiten 2 formas de crear un proyecto en Vue ClI

  1. Mediante comandos en consola
$ vue create [nombre del proyecto]
  1. Mendiante la Interfaz Grafica de Usuario (ui)
$ vue ui

Para correr nuestro proyecto en un servidor local ejecutaremos el siguiente comando

$ npm run serve

Estas dos formas de crear el proyecto lo veremos en el caso práctico con más detalle, así como crear componentes y consumir una API REST usando AXIOS.
